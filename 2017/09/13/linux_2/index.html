linux下各种编程语言安装

Posted on 2017-09-13 | Edited on 2018-01-30 | In linux

linux为Debian系列，包管理为deb

java ：终端输入

下载jdk后用文件安装，下载jdk文件后，解压，然后利用管理员权限去编辑 /etc/profile文件，加入下面这些代码

JAVA_HOME=/usr/lib/jvm/jdk1.8.0_144
JRE_HOME=$JAVA_HOME/jre
JAVA_BIN=$JAVA_HOME/bin
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export JAVA_HOME JRE_HOME PATH CLASSPATH

其中，java——home是你解压后jdk文件夹所在的路径。之后注销系统，重新登录
在终端输入：java -version看是否有输出相应的java版本信息

Picked up _JAVA_OPTIONS: -Dawt.useSystemAAFontSettings=gasp
java version "1.8.0_144"
Java(TM) SE Runtime Environment (build 1.8.0_144-b01)
Java HotSpot(TM) 64-Bit Server VM (build 25.144-b01, mixed mode)

mysql： 终端输入

sudo apt-get install mysql-server
apt-get isntall mysql-client
sudo apt-get install libmysqlclient-dev

记得期间要输入登录密码，之后在终端输入mysql -uroot -p(你设定的密码)，回车看是否登录成功

go 下载语言包进行安装

下载好go语言的包后解压（sources code）
找到你解压后的文件路径
找到/etc/profile文件
加上这行代码export PATH=$PATH:/usr/local/go/bin
终端输入go version查看go版本检验安装是否成功

go version go1.9 linux/amd64

Gradle （java包管理，广泛用于安卓，我个人也比较喜欢）

下载gradle安装包文件 （sources code）
同样，解压在某个文件夹下
找到你所解压的文件路径
/etc/profile文件加上这两行代码

export GRADLE_HOME=/home/chuntaojun/gradle-4.0.1
export PATH=$GRADLE_HOME/bin:$PATH

终端输入 gradle 查看是否出现
