什么是令牌桶算法

令牌桶算法是目前流行的用于接口调用流量限制的一种算法；大致的思路是有一个桶，用它来存储令牌，而这个桶的容量不是无限大，而是有一个最大值，同时，令牌装入桶中的速率是恒定的（个别令牌桶的实现其令牌装入速率是变动的）。每当一个请求进入时，首先查看令牌桶中是否存在令牌，如果令牌不存在，则抛弃该请求；如果令牌桶中存在令牌，则桶中的令牌数量和请求的body的大小进行比较，如果剩余的令牌桶数量能够满足本次请求的body大小，则该请求通过，并且相应的减少令牌桶中令牌的数量，否则抛弃本次请求。
